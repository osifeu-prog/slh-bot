# SLH Reputation Plan

## Goal
Build reputation before token economy.

## Principles
- Free access first.
- Contribution creates reputation.
- Reputation is earned, not purchased.

## Reputation Sources

Learning:
- Completing lessons
- Asking useful questions
- Improving knowledge

Helping:
- Answering users
- Reviewing content
- Supporting community

Creating:
- Building tools
- Writing content
- Improving system

Testing:
- Finding bugs
- Reporting issues
- Verifying features

## Rules
- No reputation farming.
- Agents verify important actions.
- Reputation history is stored.

## Future
Reputation may influence:
- SLH rewards
- Access levels
- Community roles
