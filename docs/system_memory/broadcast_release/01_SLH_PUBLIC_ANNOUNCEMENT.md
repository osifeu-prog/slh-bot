# SLH Public Announcement

SLH מתרחבת.

אנחנו בונים אקוסיסטם טכנולוגי שמחבר בין יזמים, מפתחים ושותפים כדי ליצור שירותים דיגיטליים חדשים בצורה מאובטחת ומבוססת שיתוף פעולה.

החזון:

לאפשר לאנשים מוכשרים לבנות מעל תשתית חזקה, עם כלים, ידע וסביבה שמאפשרת חדשנות לאורך זמן.

בקרוב:

- SLH Academy למפתחי Microservices
- מסלול הכשרה למפתחים
- אינטגרציות מבוססות API
- הזדמנויות לשותפים טכנולוגיים

הגישה שלנו:

חדשנות.
שיתוף פעולה.
אבטחה.
בנייה לטווח ארוך.

מתאים למפתחים, יזמים ושותפים שרוצים להיות חלק מהדור הבא של שירותים דיגיטליים.
