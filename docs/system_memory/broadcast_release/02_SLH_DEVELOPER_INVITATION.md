# SLH Developer Invitation

SLH מחפשת מפתחים שרוצים לבנות שירותים חדשים בתוך אקוסיסטם מתפתח.

המסלול כולל:

- יסודות Linux
- Python Backend
- פיתוח Web
- APIs
- בסיסי נתונים
- Microservices
- אינטגרציות מערכת

המפתחים יעבדו במודל:

למידה
 ->
פיתוח
 ->
Sandbox
 ->
אינטגרציה


המטרה:

ליצור רשת של בונים ומפתחים שיוצרים שירותים תואמים לאקוסיסטם SLH.

מתאים למי שרוצה:

- ללמוד ארכיטקטורות מודרניות
- לבנות מוצרים אמיתיים
- להשתלב בפרויקטים עתידיים
