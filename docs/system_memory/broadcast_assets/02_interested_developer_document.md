# SLH Academy - Developer Information

## Overview

SLH Academy is a developer program designed to train builders who want to create compatible services inside the SLH ecosystem.


## Learning Track

Topics:

- Linux and Termux foundations
- Python backend development
- Web applications
- API design
- Database architecture
- Redis and PostgreSQL concepts
- Docker
- Microservices architecture
- SLH Gateway integration


## Development Model

Developers work through:

Developer Service

        |

Sandbox

        |

Gateway APIs

        |

SLH Services


## Security Model

Developers do not receive:

- Production access
- Internal secrets
- Direct database access


All integrations follow:

REQUEST
 ->
AUDIT
 ->
VALIDATION
 ->
GATEWAY
 ->
SERVICE


## Goal

Create a certified network of developers and partners building services around SLH.
