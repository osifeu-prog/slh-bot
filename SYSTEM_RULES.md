# SLH System Rules

## Core Principle
Free ecosystem first.

## User Access
- No payment required initially.
- Learning and contribution are rewarded.
- Users can use core features freely.

## Reputation
Actions may increase reputation:
- Learning
- Helping
- Creating
- Testing
- Improving system

## Agents
Agents operate by defined roles.
No agent may modify core rules without approval.

## Token
SLH token is not required for access.
Token utilities may be added later.

## Safety
System state must be backed up before major changes.
